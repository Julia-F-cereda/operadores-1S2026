<h3>DESCRIÇÃO DO PROJETO</h3>

Este projeto foi desenvolvido com o objetivo de realizar calculos. O usuário insere as informações solicitadas e o sistema retorna automaticamente os cálculos realizados. O projeto foi criado para praticar lógica de programação utilizando JavaScript.

<h3>COMO EXECUTAR</h3>
<P>1. Baixe os arquivos do projeto.</P>
<P>2. Abra o arquivo `operadores-v3.html`.</P>
<P>3. Execute utilizando a extensão Live Server no VS Code.</P>
